Garden of Freedom v1.0

A local deep‑time engine and narrative‑state toolkit.

Installation:
1. Download Garden-of-Freedom-v1.0.tar.gz
2. Run: bash install-garden.sh
3. Use: garden, garden all, garden meta

Uninstall:
rm -rf ~/.garden
sudo rm /usr/local/bin/garden

Features:
- Multi‑layer deep‑time engines
- Unified view
- Presence loop
- Meta counter
- Full command palette
