# DIRECTORY PLACEMENT PROTOCOL (⟞⚓⧉)
### Canonical Rules for File Placement
`
This protocol ensures every artifact lives in the correct directory.
`
---
`
# 1. Modes
Place in:
- `SPEC/LAYERS/MODES/ 
`
Modes must never be placed in sublayers.
`
---
`
# 2. Currents
Place in:
- `SPEC/LAYERS/CURRENTS/<SUBLAYER>/ 
`
Sublayers include:
- CORE  
- COLLAPSE-LIMINAL  
- DIRECTIONAL  
- SOMATIC  
`
---
`
# 3. Connectors
Place in:
- `SPEC/LAYERS/CONNECTORS/<CATEGORY>/ 
`
Categories include:
- CORE  
- DECISION  
- MODE-PAIRS  
- CONTRIBUTOR  
`
---
`
# 4. Protocols
Place in:
- `SPEC/LAYERS/<LAYER>/PROTOCOLS/ 
`
Protocols must match the layer they operate on.
`
---
`
# 5. Contributor Artifacts
Place in:
- `SPEC/LAYERS/CONTRIBUTOR/ 
`
---
`
# 6. Forbidden Placements
- Never place artifacts in root directories  
- Never create new directories without structural justification  
`
---
`
# 7. Close
Confirm placement matches classification.
